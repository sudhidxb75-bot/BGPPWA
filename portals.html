Portal Structure

Customer, Seller, and Admin portals

The frontend keeps customer access visible, while seller and admin operations remain controlled and protected.

Customer Portal

Visible through the login icon in the header. Customers can signup, login, post requirements, and submit referrals.

Customer Login

Seller Portal

Hidden from the public menu. Sellers can use the direct URL provided by admin to login, post property details, and upload photo links. Admin must verify and approve before publishing.

Direct URL: seller-login.html

Admin Backend

Admin remains hidden from the frontend. Admin works from Google Sheets and Apps Script to verify, edit, approve, publish properties, create proposals, and track deals.

Access: Google Sheet backend only
